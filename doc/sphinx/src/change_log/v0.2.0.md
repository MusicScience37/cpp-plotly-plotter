# Release v0.2.0

- Figure builders to create figures with simpler APIs.
- More trace types.
  - Heatmap.
- More parameters of figures.
  - Subplots.
  - Titles of legends.
  - Annotations.
  - Log scale in axes.
  - Colors.
  - Dash of lines.
  - Animation.
  - Ratio of axes.
- Stabler implementation of PDF and PNG output.
- Support of SVG output.
- Fixed the issue that NaN and infinity causes exceptions.
